# 💱 Currency Converter

A simple and responsive Currency Converter web application built using **HTML, CSS, and JavaScript**. This project allows users to convert one currency into another using real-time exchange rates fetched from an API.

## 🚀 Features

- Convert between multiple currencies
- Real-time exchange rate updates
- User-friendly interface
- Responsive design for mobile and desktop
- Swap currencies easily
- Displays converted amount instantly

## 🛠️ Technologies Used

- **HTML5** – Structure of the application
- **CSS3** – Styling and responsive design
- **JavaScript (ES6)** – Logic and API integration
